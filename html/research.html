While browsing and stressing about the home page, which is also the first page of the project, I came through the page name "Flip Electronics". The website has a smooth video background and I fell in love with the concept. And I tried to make a video background in this project.

The next website I came across was "bunk.com". This website have a stunning transition between the naviagtion links. Eventhough the effect was very easy and I completed it immediately. This small effect made my project very clean.

The product page was one of the page that needed more attention. And surfing the net, I came across a page name "Catch Sketch Dark". This page was designed in dark theme which perfectly matched my project. This also presents a unique way of showing products to the users.

I looked up a lot of website to get idea for the contact page. And thankfully I found a layout which I liked immediately. The name of the website was "Benekey" and this layout really defined the contact page of my project.

At first, I thought footer would be an easy task to complete. I could not be more wrong. But fortunately, I found a footer using the flex property to perfection. I modified here and there but was thankful to find this footer on one of the university website named Bishops. The footer contains an extra navigation page for easy experience of the user. The footer section also contains the external links to social media of the website.
